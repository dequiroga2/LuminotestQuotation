# Proyecto Luminotest - Variables de Vercel

## Copiar el Service Account Key

### Windows (PowerShell):
```powershell
Get-Content .\serviceAccountKey.json -Raw | Set-Clipboard
```

### Mac/Linux:
```bash
cat serviceAccountKey.json | pbcopy
```

Después pega el contenido en la variable `FIREBASE_SERVICE_ACCOUNT_KEY` en Vercel.

---

## Lista de Variables para Vercel

Copia y pega esto en Settings → Environment Variables:

```
DATABASE_URL=
VITE_FIREBASE_API_KEY=
VITE_FIREBASE_AUTH_DOMAIN=
VITE_FIREBASE_PROJECT_ID=
VITE_FIREBASE_STORAGE_BUCKET=
VITE_FIREBASE_MESSAGING_SENDER_ID=
VITE_FIREBASE_APP_ID=
FIREBASE_SERVICE_ACCOUNT_KEY=
NODE_ENV=production
```

Luego rellena cada valor.
